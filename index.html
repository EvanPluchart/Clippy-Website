Questions fréquentes

Les détails qui comptent.

Une question reste sans réponse ? Ouvrez une issue sur GitHub.

Clippy est-il gratuit ? +

Oui. Clippy est gratuit et distribué sous licence MIT. Son code source peut être consulté, audité et amélioré par tout le monde.

Pourquoi l’autorisation Accessibilité est-elle demandée ? +

Elle permet à Clippy d’envoyer le raccourci de collage dans l’application que vous utilisiez après votre sélection. La surveillance et la recherche de l’historique fonctionnent localement.

Quelles versions de macOS sont compatibles ? +

Clippy nécessite macOS 14 Sonoma ou une version plus récente. L’application est conçue pour les Mac Apple Silicon et Intel.

Clippy envoie-t-il mon presse-papiers sur Internet ? +

Non. L’historique est enregistré sur votre Mac. Le site vérifie uniquement GitHub au chargement pour proposer la dernière release disponible.
